XY charts

XY Chart​

The XY Chart shows the relationship of data sets. In many cases, it is used for time series data, to show accurate time/value relations.

Bubble Chart​

Heat Map​

The Heatmap Chart visualizes the intensity of points on an XY Chart. The closer together the hotter (red) the map gets. In this example picture, we used time-series functions for the data range. Click here if you want to know more about time-series functions.

State Chart Time:​

Indicates states at certain time points.

Chart Periode:​

Indicate states over some time. From time point a to b.
