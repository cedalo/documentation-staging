GOTO¶

Jump to given cell for next step in current calculation sequence. Normally a Streamsheet is calculated from left to right and top to bottom. Using GOTO you can jump within the calculation logic.

Syntax

=GOTO(Cell)

Arguments

Name

Description

Cell

Target cell of GOTO.

Return Value

TRUE, if successful, otherwise error code.

Example

Function

Result

Comment

=GOTO(B1)

TRUE

The step stops at the GOTO() function, jumps back to B1 and starts the next step at B1.

Warning

Be careful, when using GOTO, as you can define an endless loop using this function.
