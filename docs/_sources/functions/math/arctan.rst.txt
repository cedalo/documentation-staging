ARCTAN
-----------------------------

Calculates the arctangent value of the given value

**Syntax**

=ARCTAN(Value)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Value
     - Value to evaluate arctangent for.

**Return Value**

Evaluated angle in radians.

**Example**

.. list-table::
   :widths: 45 10
   :header-rows: 1

   * - Function
     - Result
   * - =ARCTAN(0.3)
     - 0.2914568
