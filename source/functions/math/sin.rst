SIN
-----------------------------

Returns the sine value of the angle.

**Syntax**

=SIN(Angle)

**Arguments**

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Name
     - Description
   * - Angle
     - Angle in radians

**Return Value**

Evaluated sine value.

**Example**

.. list-table::
   :widths: 45 10
   :header-rows: 1

   * - Function
     - Result
   * - =SIN(PI())
     - 0
